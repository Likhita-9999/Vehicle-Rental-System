A console-based vehicle rental management system built in Java, simulating a simplified 
version of a real-world rental platform. Users can browse available scooties, motorcycles, 
and cars, agree to rental terms, and calculate rental costs based on hours booked.

## 🚀 Features
- Interactive terms & conditions agreement flow
- Three vehicle categories: **Scooty**, **Motorcycle**, **Car**
- Multiple models per category with per-hour pricing
- Input validation with retry-on-invalid-choice loops
- Real-time total cost calculation based on hours entered


## 🛠️ Tech Stack
- **Language:** Java
- **Concepts used:** Object-Oriented Programming (classes & objects), Scanner I/O, 
  control flow (loops, switch-case), input validation

## ▶️ How to Run
```bash
javac Main.java
java Main
```
